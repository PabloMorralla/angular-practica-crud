# Índice de documentación

**Audiencia:** transversal
**Última revisión:** 2026-03-27

Este directorio reúne la documentación funcional, formativa y operativa del proyecto.

- [training-guide.md](training-guide.md): marco general de la formación, alcance, tiempos orientativos e hitos

- [faq.md](faq.md): dudas frecuentes
- [submission-guide.md](submission-guide.md): criterio de entrega
- [support-model.md](support-model.md): cómo pedir ayuda y cuándo escalar
- [issues-guide.md](issues-guide.md): cómo registrar incidencias reales
- [ai-and-security-policy.md](ai-and-security-policy.md): uso de IA y pautas básicas de seguridad

## Para mentores

- [mentor-guide.md](mentor-guide.md): criterio de acompañamiento y seguimiento
- [rubric.md](rubric.md): marco común de evaluación

- [program-governance.md](program-governance.md): gobernanza y continuidad del programa
